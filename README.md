# Lab 1 - DC Motor with Series Excitation

A DC series-excited motor model in MATLAB/Simulink. Laboratory work for the course "Nonlinear Automatic Control Systems".

## 📁 Project Structure
📦 DC_motor
## 📂 Folder Description

| Folder | Description |
|-------|------------|
| `model` | Contains the Simulink motor model |
| `docs` | Documentation and reports |

## ⚙️ Main Files

- **`DC_motor.slx`** — main simulation model

## 🚀 Quick Start
1. Open MATLAB
2. Navigate to the model folder
3. Double-click dc_motor.slx
4. Click Run
5. Open the plots
